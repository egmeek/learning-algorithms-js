# learning-algorithms-js
Introductory machine learning algorithms demonstrated in JavaScript.
* [Perceptron demo by learning logical operations.](http://reesington.github.io/learning-algorithms-js/perceptron.html)
* [Bidirectional associative memory (BAM) demo by simple character recognition.](http://reesington.github.io/learning-algorithms-js/bam.html)
* [Self-organizing map (SOM) demo by categorizing random input in a graph.](http://reesington.github.io/learning-algorithms-js/som.html)

## Usage
Go to any of the above links and enjoy!
